**Sequences** represent an ordered collection of [[Object]]s. A sequence's [[Object]]s can be accessed using a numeric **index**. In many other languages, the array is the best-known sequence [[Type]], but Python uses [[List]]s to fill the same role. However, [[Tuple]]s and [[Range]]s are also important [[Type]]s.

They all provide [[Indexed key access]].

[[List]]s and [[Tuple]]s are said to be **heterogenous** because they can contain different [[Type]]s of [[Object]]s whereas [[Range]] is **homogeneous** because it can only contain [[Integer]]s.
