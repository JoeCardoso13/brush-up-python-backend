They're not actually an operator, but an [[Expression]] (their official name is conditional expression but that's confusing since we also have [[Conditional expression]]s).

They have the following syntax:
```
value1 if condition else value2
```

Many Python programmers dislike ternary [[Expression]]s since they are hard to read. If you decide that you like ternary [[Expression]]s, that's fine. However, use them judiciously.
