Instance variables keep track of information about the of an [[Object]].

Do not confuse them with [[Attribute]]s or [[Property]]:
- Instance variables are [[Variable]]s that are tied to an instance of a [[Class]].
- [[Attribute]]s include all instance variables and [[Instance method]]s.
- [[Property]] are a special kind of [[Method]] that enables syntax that makes the [[Property]] look like an instance variable. You can use the [[Property]] name like a [[Variable]] name. [[Property]] are usually associated with instance variables but can also be dynamically computed.
