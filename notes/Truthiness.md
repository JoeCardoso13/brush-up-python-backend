In Python the following values are said to be falsy:
- `False`, `None`
- all numeric `0` values ([[Integer]]s, [[Float]]s, complex)
- empty [[String]]s: `''`
- empty [[Collection]]s: `[]`, `()`, `{}`, `set()`, `frozenset()`, and `range(0)`
- Custom data types can also define additional falsy value(s).
All other values are truthy.
