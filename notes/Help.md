For use within REPL.

When used without arguments, the `help` function prints some basic help on how to use `help`, then leaves you in a special help mode that uses a `help>` prompt. From the `help>` prompt, you can request help on [[Module]] names, Python keywords, [[Built-in function]]s, [[Class]] names, etc. Type the appropriate words at the `help>` prompt and press {Return} or {Enter}.

(As of Python 3.13.0, you no longer need to use write `help()` to access the help system. You can, instead, just write `help`)
