It's when you use a [[Function]] to transform one [[Type]] into another. Very commonly used for [[String]]s to become numbers:
```python
print(int('5'))               # 5
print(float('3.141592'))      # 3.141592
```

You can also convert pretty much any Python value into a [[String]] with the `str` [[Function]]. The `repr` does something similar but a bit more lower-level.
