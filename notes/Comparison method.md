Here are the [[Magic method]]s that correspond to these operators:

| Operator | Method   | Description              |
| -------- | -------- | ------------------------ |
| `==`     | `__eq__` | Equal to                 |
| `!=`     | `__ne__` | Not equal to             |
| `<`      | `__lt__` | Less than                |
| `<=`     | `__le__` | Less than or equal to    |
| `>`      | `__gt__` | Greater than             |
| `>=`     | `__ge__` | Greater than or equal to |
